## Как работает программа

### Генератор

Файл:

```text
generator/generate.py
```

Создает файл:

```text
data/data.csv
```

В CSV записываются данные про горы:

* название
* страна
* высота
* сложность

---

### Аналитик

Файл:

```text
reporter/report.js
```

Читает CSV-файл и создает HTML-отчет.

Создается файл:

```text
data/report.html
```

В отчете показывается:

* количество строк
* статистика по высоте
* статистика по странам
* статистика по сложности

---

## Docker

Для генератора и аналитика используются отдельные контейнеры.

---

## Команды

Собрать генератор:

```bash
./run.sh build_generator
```

Запустить генератор:

```bash
./run.sh run_generator
```

Собрать аналитик:

```bash
./run.sh build_reporter
```

Запустить аналитик:

```bash
./run.sh run_reporter
```

Очистить данные:

```bash
./run.sh clear_data
```

---

## Результат

После запуска в папке `data` появляются файлы:

```text
data.csv
report.html
```
